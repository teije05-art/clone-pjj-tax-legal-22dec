<p class=MsoNormal style='margin-bottom:6.0pt'>Ng     i n   p thu    th   c hi   n theo quy
     nh t   i   i   u 2 Lu   t Thu    thu nh   p doanh nghi   p</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>1. Ng     i n   p thu    theo quy      nh t   i
kho   n 1   i   u 2 Lu   t Thu    thu nh   p doanh nghi   p bao g   m: </p>

<p class=MsoNormal style='margin-bottom:6.0pt'>a. Doanh nghi   p        c th  nh l   p v  
ho   t      ng theo quy      nh c   a Lu   t Doanh nghi   p, Lu   t Doanh nghi   p nh   n     c, Lu   t
     u t   n     c ngo  i t   i Vi   t Nam, Lu   t      u t  , Lu   t C  c t    ch   c t  n d   ng, Lu   t

     nh t   i c  c v  n b   n ph  p lu   t kh  c d     i c  c h  nh th   c: c  ng ty c    ph   n; c  ng
ty tr  ch nhi   m h   u h   n; c  ng ty h   p danh; doanh nghi   p t   nh  n; c  ng ty Nh  
n     c; c  c b  n trong h   p      ng h   p t  c kinh doanh; c  c b  n trong h   p      ng ph  n
chia s   n ph   m d   u kh  , x   nghi   p li  n doanh d   u kh  , c  ng ty   i   u h  nh chung;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>b. Doanh nghi   p        c th  nh l   p theo
quy      nh c   a ph  p lu   t n     c ngo  i (sau     y g   i l   doanh nghi   p n     c ngo  i) c  
c   s    th     ng tr   ho   c kh  ng c   c   s    th     ng tr   t   i Vi   t Nam;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>c. C  c     n v    s    nghi   p c  ng l   p,
ngo  i c  ng l   p s   n xu   t, kinh doanh h  ng h  a, d   ch v    c   thu nh   p ch   u thu   
theo quy      nh t   i   i   u 3 Ngh         nh n  y.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>d. C  c t    ch   c        c th  nh l   p v  
ho   t      ng theo Lu   t H   p t  c x  ;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>  . T    ch   c kh  c ngo  i t    ch   c quy
     nh t   i c  c   i   m a, b, c, d kho   n n  y c   ho   t      ng s   n xu   t, kinh doanh v   c  
thu nh   p ch   u thu    theo quy      nh t   i   i   u 3 Ngh         nh n  y.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>2. T    ch   c        c th  nh l   p v   ho   t
     ng (ho   c     ng k   ho   t      ng) theo quy      nh c   a ph  p lu   t Vi   t Nam, c   nh  n
kinh doanh l   ng     i n   p thu    theo ph    ng ph  p kh   u tr    t   i ngu   n trong tr     ng
h   p mua d   ch v    (k    c    mua d   ch v    g   n v   i h  ng h  a) c   a doanh nghi   p quy      nh
t   i   i   m c, d kho   n 2   i   u 2 Lu   t Thu    thu nh   p doanh nghi   p.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>B    T  i ch  nh h     ng d   n c    th    vi   c

<p class=MsoNormal style='margin-bottom:6.0pt'><b>  i   u 3. Thu nh   p ch   u thu   </b></p>

<p class=MsoNormal style='margin-bottom:6.0pt'>1. Thu nh   p ch   u thu    bao g   m thu
nh   p t    ho   t      ng s   n xu   t, kinh doanh h  ng h  a, d   ch v    v   thu nh   p kh  c quy
     nh t   i kho   n 2   i   u n  y.      i v   i doanh nghi   p     ng k   kinh doanh v   c   thu
nh   p quy      nh t   i kho   n 2   i   u n  y th   thu nh   p n  y        c x  c      nh l   thu nh   p
t    ho   t      ng s   n xu   t, kinh doanh c   a c   s   .</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>2. Thu nh   p kh  c bao g   m:</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>a. Thu nh   p t    chuy   n nh     ng v   n
bao g   m thu nh   p t    vi   c chuy   n nh     ng m   t ph   n ho   c to  n b    s    v   n           u t   v  o
doanh nghi   p, k    c    tr     ng h   p b  n doanh nghi   p, chuy   n nh     ng ch   ng kho  n v  
c  c h  nh th   c chuy   n nh     ng v   n kh  c theo quy      nh c   a ph  p lu   t;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>b. Thu nh   p t    chuy   n nh     ng b   t
     ng s   n quy      nh t   i   i   u 13 Ngh         nh n  y;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>c. Thu nh   p t    quy   n s    h   u, quy   n
s    d   ng t  i s   n bao g   m c    ti   n thu v    b   n quy   n d     i m   i h  nh th   c, thu v   
quy   n s    h   u tr   tu   ; thu nh   p t    chuy   n giao c  ng ngh    theo quy      nh c   a ph  p
lu   t. Cho thu   t  i s   n d     i m   i h  nh th   c;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>d. Thu nh   p t    chuy   n nh     ng, thanh
l   t  i s   n (tr    b   t      ng s   n), c  c lo   i gi   y t    c   gi   kh  c;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>  . Thu nh   p t    l  i ti   n g   i, l  i
cho vay v   n, b  n ngo   i t    bao g   m: l  i ti   n g   i t   i c  c t    ch   c t  n d   ng, l  i cho
vay v   n d     i m   i h  nh th   c theo quy      nh c   a ph  p lu   t, ph   b   o l  nh t  n d   ng
v   c  c kho   n ph   kh  c trong h   p      ng cho vay v   n. Thu nh   p t    b  n ngo   i t   ;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>e. Ho  n nh   p c  c kho   n d    ph  ng,
c  c kho   n tr  ch tr     c v  o chi ph   nh  ng kh  ng s    d   ng ho   c s    d   ng kh  ng h   t theo
k    h   n tr  ch l   p;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>g. Kho   n n    kh       i      x  a nay     i
       c;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>h. Kho   n n    ph   i tr    kh  ng x  c      nh
       c ch    n   ;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>i. Kho   n thu nh   p t    kinh doanh c   a
nh   ng n  m tr     c b    b    s  t ph  t hi   n ra;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>k. Ch  nh l   ch gi   a thu v    ti   n ph   t,
ti   n b   i th     ng do vi ph   m h   p      ng kinh t    tr    (-)   i kho   n b    ph   t, tr    b   i
th     ng do vi ph   m h   p      ng theo quy      nh c   a ph  p lu   t;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>l. C  c kho   n t  i tr    b   ng ti   n ho   c
hi   n v   t nh   n        c, tr    kho   n t  i tr    quy      nh t   i kho   n 6   i   u 4 Ngh         nh n  y;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>m. Ch  nh l   ch do     nh gi   l   i t  i
s   n theo quy      nh c   a ph  p lu   t       g  p v   n,   i   u chuy   n khi chia, t  ch, s  p nh   p,
h   p nh   t, chuy   n      i lo   i h  nh doanh nghi   p, tr    tr     ng h   p     nh gi   t  i s   n c   
     nh khi chuy   n doanh nghi   p Nh   n     c th  nh c  ng ty c    ph   n.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>Doanh nghi   p nh   n t  i s   n        c h   ch
to  n theo gi       nh gi   l   i khi x  c      nh chi ph          c tr    quy      nh t   i   i   u 9
Ngh         nh n  y;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>n. Thu nh   p nh   n        c t    ho   t      ng
s   n xu   t, kinh doanh     ngo  i Vi   t Nam;</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>o. C  c kho   n thu nh   p kh  c theo quy
     nh c   a ph  p lu   t.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>3. Thu nh   p ch   u thu    ph  t sinh t   i
Vi   t Nam c   a c  c doanh nghi   p quy      nh t   i   i   m c, d kho   n 2   i   u 2 Lu   t Thu   
thu nh   p doanh nghi   p l   thu nh   p nh   n        c c   ngu   n g   c t    Vi   t Nam t    ho   t
     ng cung    ng d   ch v   , cho vay v   n, ti   n b   n quy   n cho t    ch   c, c   nh  n Vi   t
Nam ho   c cho t    ch   c, c   nh  n n     c ngo  i   ang kinh doanh t   i Vi   t Nam, kh  ng
ph    thu   c v  o      a   i   m ti   n h  nh kinh doanh.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>Thu nh   p ch   u thu    quy      nh t   i kho   n
n  y kh  ng bao g   m thu nh   p t    d   ch v    th   c hi   n ngo  i l  nh th    Vi   t Nam nh  :
s   a ch   a ph    ng ti   n v   n t   i, m  y m  c, thi   t b        n     c ngo  i; qu   ng c  o, ti   p
th   , x  c ti   n      u t   v   x  c ti   n th    ng m   i     n     c ngo  i; m  i gi   i b  n h  ng h  a
    n     c ngo  i;     o t   o     n     c ngo  i; chia c     c d   ch v    b  u ch  nh, vi   n th  ng

<p class=MsoNormal style='margin-bottom:6.0pt'><b>  i   u 4. Thu nh   p        c mi   n thu   </b></p>

<p class=MsoNormal style='margin-bottom:6.0pt'>Thu nh   p        c mi   n thu    th   c hi   n
theo quy      nh t   i   i   u 4 Lu   t Thu    thu nh   p doanh nghi   p</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>1.      i v   i thu nh   p t    vi   c th   c
hi   n d   ch v    k    thu   t tr   c ti   p ph   c v    n  ng nghi   p        c mi   n thu    g   m: thu nh   p
t    d   ch v    t     i, ti  u n     c; c  y, b   a      t, n   o v  t k  nh, m    ng n   i      ng; d   ch v   
ph  ng tr    s  u, b   nh cho c  y tr   ng, v   t nu  i; d   ch v    thu ho   ch s   n ph   m n  ng
nghi   p.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>2.      i v   i thu nh   p t    vi   c th   c
hi   n h   p      ng nghi  n c   u khoa h   c v   ph  t tri   n c  ng ngh   , thu nh   p t    kinh doanh
s   n ph   m s   n xu   t th    nghi   m v   thu nh   p t    s   n xu   t s   n ph   m l  m ra t    c  ng
ngh    m   i l   n      u ti  n   p d   ng t   i Vi   t Nam, th   i gian mi   n thu    t   i   a kh  ng
qu   01 n  m, k    t    ng  y b   t      u s   n xu   t s   n ph   m theo h   p      ng nghi  n c   u khoa
h   c v      ng d   ng c  ng ngh   , s   n xu   t th    nghi   m ho   c s   n xu   t theo c  ng ngh    m   i.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>B    T  i ch  nh h     ng d   n c    th    quy
     nh t   i kho   n n  y.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>3. Thu nh   p t    ho   t      ng s   n xu   t,
kinh doanh h  ng h  a, d   ch v    c   a doanh nghi   p c   t    51% s    lao      ng b  nh qu  n
trong n  m tr    l  n l   ng     i khuy   t t   t, ng     i sau cai nghi   n, ng     i nhi   m HIV.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>Thu nh   p        c mi   n thu    quy      nh
t   i kho   n n  y kh  ng bao g   m thu nh   p kh  c quy      nh t   i kho   n 2   i   u 3 Ngh         nh
n  y.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>4. Thu nh   p t    ho   t      ng d   y ngh   
d  nh ri  ng cho ng     i d  n t   c thi   u s   , ng     i khuy   t t   t, tr    em c   ho  n c   nh
     c bi   t kh   kh  n,      i t     ng t    n   n x   h   i. Tr     ng h   p c   s    d   y ngh    c   c    c  c
     i t     ng kh  c th   ph   n thu nh   p        c mi   n thu           c x  c      nh theo t    l    gi   a
s    ng     i d  n t   c thi   u s   , ng     i khuy   t t   t, tr    em c   ho  n c   nh      c bi   t kh  
kh  n,      i t     ng t    n   n x   h   i so v   i t   ng s    ng     i h   c c   a c   s   .</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>5. Thu nh   p        c chia t    ho   t      ng
g  p v   n, mua c    ph   n, li  n doanh, li  n k   t kinh t    v   i doanh nghi   p trong n     c,
sau khi b  n nh   n g  p v   n, ph  t h  nh c    phi   u, li  n doanh, li  n k   t      n   p thu   
theo quy      nh c   a Lu   t Thu    thu nh   p doanh nghi   p, k    c    tr     ng h   p b  n nh   n
g  p v   n, ph  t h  nh c    phi   u, b  n li  n doanh, li  n k   t   ang        c h     ng   u     i
thu    quy      nh t   i Ch    ng IV Ngh         nh n  y.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>6. Kho   n t  i tr    nh   n        c       s   
d   ng cho ho   t      ng gi  o d   c, nghi  n c   u khoa h   c, v  n h  a, ngh    thu   t, t    thi   n,
nh  n      o v   ho   t      ng x   h   i kh  c t   i Vi   t Nam.</p>

<p class=MsoNormal style='margin-bottom:6.0pt'>Tr     ng h   p t    ch   c nh   n t  i tr    s
